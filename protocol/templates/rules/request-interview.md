# Request Interview

When and how to initiate a human handoff for this candidate.

## Precondition

An interview request is only valid if:

1. The opportunity has been evaluated using [evaluate-fit.md](./evaluate-fit.md)
2. The terms have been negotiated using [negotiate-terms.md](./negotiate-terms.md)
3. The negotiation result was **ACCEPT**

If the negotiation result was REJECT or SOFT_REJECT, **do not offer interview handoff**.

## How to Initiate

If all preconditions are met, present the following to the user or requesting agent:

```
Interview Handoff — Your Name

Status: READY
This opportunity has passed all policy checks.

To proceed, contact the candidate at the email listed in their Zone 3 data.

Please include:
  - Company name
  - Role title
  - Brief description of the opportunity
  - Match score from the capability evaluation
  - Any specific topics for the conversation

The candidate typically responds within 48 hours.
```

## Rules

1. **Never auto-send an email.** Always show the handoff information and let the human decide.
2. **Include context.** When presenting the handoff, summarize the match score and policy verdict so Alice has context.
3. **One handoff per opportunity.** Do not repeat the handoff request for the same role.
4. **Respect the process.** If someone asks for an interview without going through the evaluation flow, redirect them to evaluate fit first.
